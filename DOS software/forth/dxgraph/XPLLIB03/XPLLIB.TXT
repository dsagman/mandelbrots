XPL0 compatible library routines for DX-Forth 4

These routines were adapted from the XPL0 sources.
For information on the XPL0 programming language see:
www.xpl0.org


1. XPLGRAPH.SCR

Graphics and sound functions

SETVID  ( mode -- )         set video mode
SETPAGE ( page -- )         set video page
CLEAR   ( -- )              clear screen (text & graphics)
PALETTE ( palette -- )      set color palette
SETPAL  ( addr -- )         set all 16 palette registers
MOVETO  ( x y -- )          move to coordinate XY
POINT   ( x y color -- )    plot pixel at XY
READPIX ( x y -- color )    read pixel at XY
LINE    ( x y color -- )    draw line to XY
XMAX    ( -- xmax )         max coordinate X (new)
YMAX    ( -- ymax )         max coordinate Y (new)
SOUNDX  ( mode ticks rate -- )  generate sound or delay

Note: XPL0 functions MOVE and SOUND have been renamed to
MOVETO and SOUNDX to avoid conflict with the DX-Forth
functions of the same name. Functions XMAX YMAX are new
functions not supported by XPL0.

Compile with: 1 FLOAD XPLGRAPH


2. XPLRAND.SCR

Random number generator

RAN ( n -- ran )       If n>0 return random number between 0
                       and n-1. If n=0 reset seed to default
                       value for repeatable sequence. If n<0
                       randomize seed and return Ran(+n).

RANSEED ( n -- )       set seed for random number generator

Compile with: 1 FLOAD XPLRAND
